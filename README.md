## О проекте

**Гео-D7** — это современный веб-портал, посвящённый актуальным вопросам геодезии, геодинамики и спутниковой навигации. Проект предоставляет доступ к каталогу статей, словарю терминов, аналитическим материалам, обзорам оборудования, а также информации о партнёрских организациях и полезных источниках.

### Ключевые особенности

| Особенность | Описание |
|-------------|----------|
| **SSR/SSG** | Серверный рендеринг для максимальной SEO-оптимизации |
| **Адаптивный дизайн** | Корректное отображение на всех устройствах |
| **Аутентификация** | Полноценная система регистрации и авторизации через Supabase |
| **PDF просмотрщик** | Встроенный просмотр PDF-файлов с поддержкой масштабирования |
| **Yandex Cloud Storage** | Надёжное хранение и доставка PDF-документов |
| **Интернационализация** | Поддержка русского и английского языков |
| **SEO оптимизация** | Мета-теги, Open Graph, JSON-LD, sitemap.xml |
| **Система просмотров** | Учёт популярности статей с защитой от накрутки |
| **Партнёрская сеть** | Информация об организациях IGS, ITRF, EUSPA, ESA и др. |
| **Полезные источники** | Ссылки на стандарты ISO, ГОСТ, проекты и сервисы PPP |

## Архитектура

### Технологический стек

| Технология | Назначение |
|------------|------------|
| **Next.js 16 (App Router)** | Фреймворк — серверный рендеринг, маршрутизация |
| **React 19** | UI библиотека |
| **TypeScript** | Типизация и безопасность кода |
| **Supabase** | База данных, аутентификация |
| **Yandex Cloud Storage** | Хранение и доставка PDF-файлов |

## API и интеграции

### **Supabase (База данных)**

```sql
-- Основные таблицы
articles        # Статьи (id, title, pdf_path, category, views, likes)
sections        # Секции статей
article_sections # Связь статей и секций
users           # Пользователи (через Supabase Auth)
partners        #Партнёрские организации (IGS, ITRF, EUSPA, ESA...)
sources        #Полезные источники и ссылки (ISO, ГОСТ, проекты, сервисы PPP)
```

## Структура проекта

```bash
src/
├── app/                          # App Router (страницы)
│   ├── content/                  # Динамические страницы статей
│   ├── dictionary/               # Словарь терминов
│   ├── profile/                  # Личный кабинет
│   ├── metadata/                 # Метаданные и SEO
│   └── layout.tsx                # Корневой layout
│
├── components/                   # React компоненты
│   ├── ui/                       # Базовые UI компоненты
│   └── shared/                   # Бизнес-компоненты
│
├── lib/                          # Утилиты и сервисы
│   ├── supabase/                 # Supabase клиенты
│   ├── services/                 # Бизнес-логика
│   ├── i18n/                     # Интернационализация
│   └── auth/                     # Авторизация
│
├── contexts/                     # React контексты
├── hooks/                        # Кастомные хуки
├── styles/                       # Глобальные стили
├── types/                        # TypeScript типы
└── locales/                      # Переводы (ru, en)
```
